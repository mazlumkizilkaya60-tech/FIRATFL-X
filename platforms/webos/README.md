# LG webOS

Deneme icin iki yol vardir:

- `tv-build/webos-packaged`
- `tv-build/webos-hosted-lab`

Oneri:

- Store ve cihaz testi icin `webos-packaged`
- Hızli hosted URL denemesi icin `webos-hosted-lab`

Notlar:

- `appinfo.json` otomatik uretilir.
- webOS CLI veya webOS Studio ile package/install yapabilirsiniz.
- Hosted shell, `tv.config.json` icindeki `hostedAppUrl` adresine redirect eder.
