# Samsung Tizen

Deneme icin iki yol vardir:

- `tv-build/tizen-packaged`
- `tv-build/tizen-hosted-lab`

Oneri:

- Gercek TV denemesi icin `tizen-packaged`
- Hızli laboratuvar testi icin `tizen-hosted-lab`

Notlar:

- Samsung tarafinda hosted app modeli vardir ama genel dagitim ve API erisimi icin packaged app daha dogrudur.
- `config.xml` otomatik uretilir.
- Uygulamayi Tizen Studio ile package edip TV Developer Mode uzerinden yukleyin.
